## Clone git 
https://github.com/9Sof/Vaccine-Covid19

## What is Project using
#### node
#### postgreSQL

## Install NestJS

```bash
$ npm i -g @nestjs/clinpm i -g @nestjs/cli
```

## Installation

```bash
$ npm install
```

## Create Database
Create Database on Postgres. Name database "vaccine_covid19"
Host: localhose
Post: 5432

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
